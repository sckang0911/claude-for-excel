# L601｜應收帳款有沒有漏記

<div className="lesson-meta">

## 📖 情境說明

月底稽核發現，系統裡的發票記錄比銀行實際收款少了幾筆。可能是有收款但忘記開發票，也可能是發票開了但還沒收款。

---

## 🎯 學習目標

交叉比對發票記錄和收款記錄，找出漏記的項目。

## 🤖 Claude 指令範本

複製以下指令，貼入 Excel 的 Claude 側邊欄：

<div className="claude-prompt">有兩張工作表：
- 「發票清單」：發票號碼、客戶、金額、開立日期
- 「收款記錄」：收款日期、客戶、金額、發票號碼（可能空白）

請幫我做完整的稽核：
1. 有發票但無對應收款記錄的有哪些？（可能是應收未收）
2. 有收款但找不到對應發票的有哪些？（可能是漏開發票）
3. 發票金額和收款金額不一致的有哪些？
4. 請整理成一份稽核差異表，說明每個差異的可能原因。</div>

---

## 💡 進階技巧

- 建議先以「發票號碼」為比對鍵，再用「客戶+金額」做第二次比對
