# L604｜報帳稽核缺了哪些憑證

<div className="lesson-meta">

## 📖 情境說明

稽核人員要查核費用報帳，確認每一筆費用都有附上對應的憑證（收據或發票）。需要找出哪些報帳記錄缺少憑證。

---

## 🎯 學習目標

比對報帳清單和憑證清單，找出缺少憑證的報帳記錄。

## 🤖 Claude 指令範本

複製以下指令，貼入 Excel 的 Claude 側邊欄：

<div className="claude-prompt">

兩個工作表：
- 「報帳清單」：報帳單號、日期、部門、項目、金額
- 「憑證清單」：憑證號碼、對應報帳單號、憑證類型（收據/發票）

請幫我：
1. 找出「報帳清單」中沒有對應憑證的記錄（缺件）
2. 找出同一張報帳單對應多張憑證的記錄（可能重複報帳）
3. 依部門統計，哪個部門缺件最多？
4. 缺件金額合計是多少？
請整理成稽核報告格式。

</div>
