# L204｜報帳金額含符號無法計算

<div className="lesson-meta">
  <span>💼 費用預算</span>
  <span>⏱ 30 分鐘</span>
  <span>🏷 入門</span>
</div>

## 📖 情境說明

財務部收到各部門的報帳表格，發現金額欄裡混入了各種奇怪的格式：有的寫「NT$1,200」、有的寫「1200元」、有的寫「$1,200.00」，根本沒辦法直接加總。

---

## 🎯 學習目標

找出金額欄中格式不一致的記錄，並了解有哪些不同格式。

## 🤖 Claude 指令範本

複製以下指令，貼入 Excel 的 Claude 側邊欄：

<div className="claude-prompt">這是費用報帳明細表，金額欄的格式不統一，有些含有貨幣符號或文字。
請幫我：
1. 列出金額欄中所有不同的格式類型（例如：純數字、含$、含NT$、含「元」字等）
2. 找出格式有問題、無法直接計算的記錄
3. 建議我怎麼統一這些格式
4. 如果你能直接讀取，請告訴我清理後的金額總計應該是多少</div>
