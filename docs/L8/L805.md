## 🤖 Claude 指令範本

複製以下指令，貼入 Excel 的 Claude 側邊欄：

<div className="claude-prompt">

業務財務資料：
- 固定成本：每月80萬元（薪資、租金、設備折舊）
- 變動成本：每收入1元花費0.45元（材料、服務成本）
- 目前月收入：65萬元，每月增長率：約8%

請幫我建立損益預測模型：
1. 損益兩平收入公式：固定成本 ÷ (1 - 變動成本率)
2. 預測未來12個月的月收入（用等比增長公式）
3. 計算每月的利潤/虧損
4. 用 Excel 找出損益兩平的月份（哪個月第一次轉虧為盈）
5. 製作一個損益預測圖，顯示收入線和成本線的交叉點
請同時說明如果增長率降到5%，損益兩平點會延後幾個月。

</div>
