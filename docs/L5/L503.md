# L503｜勞健保級距自動帶入薪資表

<div className="lesson-meta">
  <span>👥 薪資人資</span>
  <span>⏱ 40 分鐘</span>

</div>

## 📖 情境說明

薪資組需要根據員工薪資，自動帶入正確的勞健保投保級距，再計算員工和公司各應負擔的保費金額。

---

## 🎯 學習目標

用 VLOOKUP 從投保級距對照表自動帶入薪資對應的保費。

## 🤖 Claude 指令範本

複製以下指令，貼入 Excel 的 Claude 側邊欄：

<div className="claude-prompt">

「員工薪資」工作表有：員工姓名（A欄）、月薪（B欄）
「級距表」工作表有：最低薪資（A欄）、投保薪資（B欄）、健保員工負擔（C欄）、勞保員工負擔（D欄）

請幫我在「員工薪資」工作表寫公式：
1. C欄：用 VLOOKUP 找出對應的投保薪資（近似比對）
2. D欄：健保費（員工負擔）
3. E欄：勞保費（員工負擔）
4. F欄：健保費（公司負擔，佔比約60%）

請說明為什麼這裡要用「近似比對」而不是「完全比對」。

</div>

---

## 💡 進階技巧

- VLOOKUP 近似比對時，級距表的第一欄必須從小到大排序，否則結果會錯
