## 📖 情境說明

年底各部門分別提交了報帳清單，財務要把5個部門的清單合併成一張總表，並加上部門名稱欄位。

## 🎯 學習目標

合併多個工作表的資料到總表，並加入來源標記。

## 🤖 Claude 指令範本

複製以下指令，貼入 Excel 的 Claude 側邊欄：

<div className="claude-prompt">

Excel 檔案裡有5個工作表，分別是「業務部」「財務部」「護理部」「行政部」「技術部」，每個工作表都有：日期、項目、金額、申請人。

請告訴我最簡單的方法，把5個工作表的資料合併到新的「總表」工作表，並且在每一行前面加上「部門」欄位（自動帶入來源工作表的名稱）。

請說明：手動貼上法 vs Power Query 法，各自的優缺點是什麼？哪種方法比較適合我？
