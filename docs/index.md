# 智齡科技 Claude for Excel｜課程總覽

> **免費自學課程** · 40堂 · 8個Level · 從零開始到財務模型  
> 製作：[智齡科技 Jubo Technology](https://jubo.health)

---

## 🗺️ 課程地圖

```
階段一｜個人操作（L1–L2）
  L1 讀  →  從表格取出答案
  L2 找  →  找出資料中的問題

階段二｜工具應用（L3–L4）
  L3 算  →  讓 Excel 自動運算
  L4 畫  →  把數字變成圖表

階段三｜整合查核（L5–L6）
  L5 合  →  跨表整合與比對
  L6 查  →  對帳、稽核、驗證

階段四｜報告決策（L7–L8）
  L7 報  →  管理報表與摘要
  L8 測  →  預測、模型、決策
```

---

## 📐 全課程一覽

| Level | 主題 | 前提能力 | 學習階段 |
|-------|------|----------|----------|
| 🟢 L1 | **讀** — 從表格讀出答案 | 不需要懂任何公式，只要會打字 | 個人操作 |
| 🟢 L2 | **找** — 找出資料中的問題 | 知道什麼是欄、列、儲存格 | 個人操作 |
| 🔵 L3 | **算** — 讓 Excel 自動運算 | 能描述「我想要什麼結果」 | 工具應用 |
| 🔵 L4 | **畫** — 把數字變成圖 | 能選定資料範圍 | 工具應用 |
| 🟠 L5 | **合** — 跨表整合與比對 | 理解不同工作表的結構 | 整合查核 |
| 🟠 L6 | **查** — 對帳、稽核、驗證 | 理解對帳邏輯 | 整合查核 |
| 🔴 L7 | **報** — 管理報表與摘要 | 理解數字彙整邏輯 | 報告決策 |
| 🔴 L8 | **測** — 預測、模型、決策分析 | 具備基本財務思維 | 報告決策 |

## 🚦 學習路徑建議

**所有人都從 L1 開始。**

完成 L1–L4（基礎20堂）後，根據你的工作職能選擇：

| 如果你是… | 建議選擇 |
|-----------|----------|
| 財務/會計日常作業 | → L5 合 + L6 查 |
| 部門主管/報告製作 | → L7 報 |
| 財務主管/CFO | → L7 報 + L8 測 |
| 倉管/採購 | → L5 合 + L6 查（L502-L602） |
| HR/薪資 | → L5 合（L503）+ L6 查（L603） |

## 💡 使用說明

1. 每堂課有 **兩個檔案**：課程網頁教材 + `.xlsx` 練習檔
2. 先讀課程網頁，了解故事背景和操作步驟
3. 下載 `.xlsx` 練習檔，搭配 **Claude 側邊欄** 練習
4. 參考 xlsx 的「**🤖 指令備忘卡**」工作表，複製 Claude 指令
