# L405｜一頁式主管Dashboard

<div className="lesson-meta">
  <span>📊 經營績效</span>
  <span>⏱ 35 分鐘</span>
  <span>🏷 基礎</span>

## 📖 情境說明

董事會月報需要一個「一頁式 Dashboard」，在同一個工作表裡顯示本月關鍵數字：收款金額、費用合計、人力成本、設備數量。讓主管不用翻多張表格。

---

## 🎯 學習目標

在 Excel 中設計一個包含多個指標的 Dashboard 頁面。

## 🤖 Claude 指令範本

複製以下指令，貼入 Excel 的 Claude 側邊欄：

<div className="claude-prompt">

我想在「Dashboard」工作表製作一個主管月報看板。
資料分散在不同工作表：
- 收款數據在「應收帳款」工作表
- 費用數據在「費用表」工作表

請說明：
1. 怎麼用 Excel 製作一個視覺化看板，包含：大數字KPI區塊、1個圓餅圖、1個長條圖
2. 如何用儲存格引用（='工作表名'!儲存格）讓 Dashboard 的數字自動連動到來源資料
3. 如何設定看板的背景色和邊框讓它看起來專業
請用繁體中文說明。

</div>

---

## 💡 進階技巧

- Dashboard 的關鍵是「連動」，數字要從來源工作表自動帶過來，不要手動輸入
- 可以在指令備忘卡工作表找到更多美化 Dashboard 的 Claude 指令
