# L305｜幫我加一個收款狀態欄

<div className="lesson-meta">
  <span>📊 經營績效</span>
  <span>⏱ 35 分鐘</span>
  <span>🏷 基礎</span>
</div>

## 📖 情境說明

業務主管想在每月業績表格中加一欄「收款狀態」，根據收款達成率自動顯示「達標」、「接近」或「未達」，方便開會時快速了解各業務員的狀況。

---

## 🎯 學習目標

用 IF/IFS 公式加入自動判斷的狀態欄。

## 🛠️ 操作步驟

<div className="step-header">步驟 1｜打開練習檔</div>

---

## 🤖 Claude 指令範本

複製以下指令，貼入 Excel 的 Claude 側邊欄：

<div className="claude-prompt">

業績表格中，F欄是「收款達成率」（百分比格式），資料從F2開始。
請幫我在G欄寫一個公式「收款狀態」：
- ≥ 100%：顯示「🏆 達標」
- 80% ~ 99%：顯示「⚡ 接近」
- &lt; 80%：顯示「❌ 未達」

請給我 IFS 和 IF 兩種寫法，並說明哪種比較適合 Excel 版本較舊的電腦。

</div>

---

## 💡 進階技巧

- IFS 函數需要 Excel 2019 以上版本，舊版請用巢狀 IF
- 門檻數字可以根據公司實際目標調整

- 將 Claude 指令範本套用到你的實際資料上
- 根據情況調整指令，獲得更精準的結果
