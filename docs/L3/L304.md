## 🤖 Claude 指令範本

複製以下指令，貼入 Excel 的 Claude 側邊欄：

<div className="claude-prompt">

費用追蹤表格結構如下：
- C欄：預算金額
- D欄：實際費用
- 資料從第2行開始

請幫我寫 Excel 公式：
1. E欄「差異金額」：實際費用 - 預算（負數表示節省，正數表示超支）
2. F欄「差異%」：差異金額 ÷ 預算，格式為百分比
3. G欄「狀態」：用 IF 公式，超支顯示「⚠️ 超支」，節省顯示「✅ 節省」，相等顯示「持平」
請附上說明讓我理解公式的邏輯。

</div>
