## 🤖 Claude 指令範本

複製以下指令，貼入 Excel 的 Claude 側邊欄：

<div className="claude-prompt">預算追蹤表：費用項目、年度預算、已使用金額、目前月份（共12個月，現在是第8個月）。

請幫我建立預算執行進度看板：
1. 計算「預算使用率」和「時間進度」（目前月份÷12）
2. 「執行狀態」判斷：使用率 > 時間進度×110% = 超支風險、使用率 < 時間進度×90% = 執行緩慢、其他 = 正常
3. 用儲存格中的重複字元（例如「██████░░░░」）製作視覺化進度條
4. 預計全年費用（已使用÷已使用月份×12），並計算是否會超出年度預算
請提供完整公式和看板設計建議。</div>

---

## 💡 進階技巧

- 用 REPT 函數重複字符可以製作文字進度條：=REPT("█",ROUND(使用率*10,0))&REPT("░",10-ROUND(使用率*10,0))
